# Proyecto-Final
> Este es mi proyecto final del ciclo DAW.
Consiste en una plataforma donde los usuarios pueden comprar y vender cartas coleccionables.
El backend está construido con Node.js, Express y una arquitectura basada en multiServicios.

# Requisitos del sistema (programas necesarios)
- Node.js (versión 18 o superior), ya viene incluido npm.
- Clonar el repositorio o descargar los archivos.
- Abrimos terminal donde tenemos el proyecto
- npm install (instalamos dependencias necesarias)
- npm start

__Si la terminal da error de ejecución de scripts: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned__
> Permite ejecutar scripts locales sin firmar
- Editor recomendado: Visual Studio Code.
